# gridpainter-backend


Ett spel där fyra spelare på tid ska rita av en slumpad bild. Spelarna kan även chatta med varandra.

------------------------------

Projektet är byggt i Node.js med express och socket.io.

Starta projektet med npm i samt nodemon start
